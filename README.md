# 🫀 Project Title: Automatic Bag-Valve Ventilator with Patient Monitoring

🔍 Overview:
This project focuses on the development of a low-cost automated ventilator system designed for emergency and resource-limited settings.
The system integrates sensor-based monitoring and adaptive control for patient support.

🎯 Objectives:
Develop an automated ambu bag compression system
Monitor patient vital parameters in real-time
Enable adaptive ventilation based on physiological conditions

⚙️ System Design:
Microcontroller: Arduino
Actuation: Servo motor-driven compression
Sensors: Heart rate, temperature
Interface: Web-based monitoring dashboard

🧪 Methodology:
Designed embedded system for real-time control
Implemented sensor data acquisition and filtering
Developed control logic for adaptive ventilation
Integrated IoT platform for remote monitoring

📊 Results:
Achieved functional automated ventilation system
Reduced cost by ~80% compared to commercial ventilators
Successfully monitored and displayed patient data in real time

📈 Key Insights:
Sensor-driven control improves responsiveness
Low-cost systems can replicate essential ventilator functions
System reliability depends on calibration and sensor accuracy

⚠️ Limitations:
Not clinically validated
Limited precision compared to ICU-grade ventilators
